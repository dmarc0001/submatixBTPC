Introduction

This program connects to the SPX42 diving computer via your PC's Bluetooth interface. It can read out configurations and set new ones.
The program is able to handle several SPX42 in parallel and saves data to the same database separately.
Furthermore, the SPX42 diving computer's logbook can be read out and saved in a local database.
The stored log files can be exported in the UDDF format for archiving or editing purposes. The UDDF format is a universal data format for exchanging diving related data.
An example would be the author's 'Diving Log' which serves as personal logbook including saving of profiles and other data.
